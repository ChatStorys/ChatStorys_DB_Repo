## DB
- DB와 관련된 코드 및 파일을 관리합니다.

### 사용법
- MongoDB Altas에 Python으로 접근하기 위해서는 아래 명령어를 실행해 드라이버를 설치해야 합니다. </br>
  python -m pip install "pymongo[srv]"   </br>
- 파이썬 버전은 4.7 이상입니다.
